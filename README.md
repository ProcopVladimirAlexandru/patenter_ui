# Patent Infringement UI

Svelte-powered UI for patent infringement detection. Very much unpolished. Hardcoded to use local backend at `localhost:8444`.

## Developing

Install dependencies with `pnpm install` then run the following command for the developing server:

```sh
pnpm run dev
```

## Building

To create a production version of your app:

```sh
pnpm run build
```
